# How do I set up and run this project?

## Getting Started

### Prerequisites

- Node.js (LTS version recommended)
- bun

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd remix-of-resale-hub-central

# Install dependencies
bun install
```

### Running

```bash
# Development
npm run dev
```
